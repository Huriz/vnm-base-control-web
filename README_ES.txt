REQUISITO PREVIO
────────────────
Necesitas tener instalado Node.js (versión 18 o superior).
Descárgalo gratis en: https://nodejs.org  (botón "LTS")
Instálalo con las opciones por defecto y reinicia el PC.

CÓMO USARLO
───────────
1. Abre el VNM SIMCENTER y asegúrate de que el servidor REST
   está activado (Settings → Enable REST Server).

2. Haz doble clic en  start.bat

   Se abrirá el navegador en http://localhost:3000
   El servidor se queda corriendo en la ventana de comandos.

3. Para cerrarlo, cierra la ventana de comandos (o Ctrl+C).

NOTAS
─────
- El servidor escucha en el puerto 3000.
- La API del VNM debe estar en localhost:9000.
- No necesita internet ni instalar nada más.
