REQUISITO
─────────
È necessario avere Node.js installato (versione 18 o superiore).
Scaricalo gratuitamente su: https://nodejs.org  (pulsante "LTS")
Installalo con le opzioni predefinite e riavvia il PC.

COME USARLO
───────────
1. Apri VNM SIMCENTER e assicurati che il server REST sia attivo
   (Settings → Enable REST Server).

2. Fai doppio clic su  start.bat

   Il browser si aprirà su http://localhost:3000
   Il server rimane in esecuzione nella finestra dei comandi.

3. Per fermarlo, chiudi la finestra dei comandi (o premi Ctrl+C).

NOTE
────
- Il server è in ascolto sulla porta 3000.
- L'API VNM deve essere in esecuzione su localhost:9000.
- Non è necessaria una connessione internet né altre installazioni.
