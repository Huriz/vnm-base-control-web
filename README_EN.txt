REQUIREMENT
───────────
You need Node.js installed (version 18 or higher).
Download it for free at: https://nodejs.org  (click "LTS")
Install with default options and restart your PC.

HOW TO USE
──────────
1. Open VNM SIMCENTER and make sure the REST server is enabled
   (Settings → Enable REST Server).

2. Double-click  start.bat

   Your browser will open at http://localhost:3000
   The server keeps running in the command window.

3. To stop it, close the command window (or press Ctrl+C).

NOTES
─────
- The server listens on port 3000.
- The VNM API must be running on localhost:9000.
- No internet connection or extra installation required.
