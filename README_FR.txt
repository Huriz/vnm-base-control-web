PRÉREQUIS
─────────
Vous devez avoir Node.js installé (version 18 ou supérieure).
Téléchargez-le gratuitement sur : https://nodejs.org  (bouton "LTS")
Installez-le avec les options par défaut et redémarrez le PC.

COMMENT L'UTILISER
──────────────────
1. Ouvrez VNM SIMCENTER et vérifiez que le serveur REST est activé
   (Settings → Enable REST Server).

2. Double-cliquez sur  start.bat

   Le navigateur s'ouvrira sur http://localhost:3000
   Le serveur continue de tourner dans la fenêtre de commandes.

3. Pour l'arrêter, fermez la fenêtre de commandes (ou appuyez sur Ctrl+C).

REMARQUES
─────────
- Le serveur écoute sur le port 3000.
- L'API VNM doit être en cours d'exécution sur localhost:9000.
- Aucune connexion internet ni installation supplémentaire requise.
