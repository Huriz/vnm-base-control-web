VORAUSSETZUNG
─────────────
Du benötigst Node.js (Version 18 oder höher).
Lade es kostenlos herunter unter: https://nodejs.org  (Schaltfläche "LTS")
Installiere es mit den Standardoptionen und starte den PC neu.

VERWENDUNG
──────────
1. Öffne VNM SIMCENTER und stelle sicher, dass der REST-Server aktiviert ist
   (Settings → Enable REST Server).

2. Doppelklicke auf  start.bat

   Der Browser öffnet sich unter http://localhost:3000
   Der Server läuft weiter im Befehlsfenster.

3. Zum Beenden schließe das Befehlsfenster (oder drücke Ctrl+C).

HINWEISE
────────
- Der Server hört auf Port 3000.
- Die VNM API muss auf localhost:9000 laufen.
- Keine Internetverbindung oder zusätzliche Installation erforderlich.
